# ブログ記事の追加方法

このディレクトリにマークダウンファイル（.md または .mdx）を追加することで、新しいブログ記事を公開できます。

## ファイル形式

ファイル名がURLのスラッグになります。例：
- `welcome.md` → `/blog/welcome`
- `zopf-interview.md` → `/blog/zopf-interview`

## フロントマター

各ファイルの先頭に、以下の形式でメタデータを記述してください：

```markdown
---
title: "記事のタイトル"
date: "2025-01-15"
author: "著者名"
excerpt: "記事の要約（2-3行程度）。ブログ一覧ページに表示されます。"
---

ここから記事本文を書きます...
```

## 記事本文の書き方

### 見出し

```markdown
## 大見出し（h2）
### 中見出し（h3）
#### 小見出し（h4）
```

### 段落

空行で段落を区切ります。

### リスト

```markdown
- 箇条書き
- 箇条書き

1. 番号付きリスト
2. 番号付きリスト
```

### 引用

```markdown
> 引用文
```

### リンク

```markdown
[リンクテキスト](https://example.com)
```

### 強調

```markdown
**太字**
*イタリック*
```

### 水平線

```markdown
---
```

## 例

詳しい例は、このディレクトリ内の既存の記事（`welcome.md`、`zopf-interview.md`、`domestic-wheat.md`）を参照してください。

## 注意事項

- 日付は `YYYY-MM-DD` 形式で記述してください
- ファイル名には日本語を使わず、英数字とハイフンのみを使用してください
- 新しい記事を追加したら、開発サーバーを再起動してください
